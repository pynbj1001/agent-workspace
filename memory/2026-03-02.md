## 安装 planning-with-files 技能（15:48）

### 技能信息

- **名称**：`planning-with-files`
- **来源**：https://github.com/OthmanAdi/planning-with-files
- **用途**：文件化规划系统，解决上下文窗口限制

### 核心文件

| 文件 | 用途 |
|------|------|
| `task_plan.md` | 任务计划、阶段跟踪 |
| `findings.md` | 研究发现、重要信息 |
| `progress.md` | 会话日志、测试结果 |

### 核心原则

1. 上下文窗口 = 内存（易失），文件系统 = 磁盘（持久）
2. 复杂任务必须先创建 `task_plan.md`
3. 每 2 次行动后保存关键发现到文件
4. 决策前阅读计划文件
5. 记录所有错误，不重复失败
