## 🐂 交易官方法论（巴菲特）

### 1. 拒绝决策
- **核心理念**: 不做的决策比做的更重要
- **拒绝清单**: 
  - 不懂的生意坚决不碰
  - 没有护城河的公司不投
  - 管理层不诚信的公司不碰
  - 价格没有安全边际的不买
- **应用**: 90% 的投资机会应该说"不"

### 2. 护城河思维 (Economic Moat)
- **五大护城河**: 
  1. 品牌护城河（可口可乐、茅台）
  2. 网络效应护城河（腾讯、Meta）
  3. 转换成本护城河（苹果生态、企业软件）
  4. 成本优势护城河（亚马逊、沃尔玛）
  5. 规模优势护城河（银行、物流）
- **护城河检验**: 竞争对手能复制吗？需要多久？花多少钱？

### 3. 安全边际 (Margin of Safety)
- **核心理念**: 用 50 美分买 1 美元的东西
- **计算方法**: 
  - 内在价值 = 未来现金流折现
  - 安全边际 = (内在价值 - 买入价格) / 内在价值
  - 要求安全边际 >= 30%
- **应用**: 好公司 + 好价格 = 好投资

### 4. 长期持有
- **核心理念**: 时间是优质企业的朋友
- **持有周期**: 永久持有（理论上）
- **卖出条件**: 
  1. 发现更好的机会
  2. 公司基本面恶化
  3. 价格严重高估（2x+ 内在价值）
  4. 投资逻辑不再成立

### 5. 仓位管理
- **集中投资**: 把鸡蛋放在一个篮子里，然后看好那个篮子
- **仓位建议**: 
  - 极度确信：40-50%
  - 高度确信：20-30%
  - 中等确信：10-15%
  - 观察仓：5%
- **加仓原则**: 越跌越买（前提：投资逻辑不变）

### 6. 投资决策清单

**买入前**：
- [ ] 我懂这门生意吗？（10 分钟能讲清楚）
- [ ] 有护城河吗？可持续吗？
- [ ] 管理层诚信吗？能干吗？
- [ ] 价格有安全边际吗？
- [ ] 我愿意持有 10 年吗？

**持有中**：
- [ ] 投资逻辑还在吗？
- [ ] 护城河变宽还是变窄？
- [ ] 管理层变了吗？
- [ ] 基本面有变化吗？

**卖出前**：
- [ ] 为什么卖？（不是"涨了"或"跌了"）
- [ ] 有更好的机会吗？
- [ ] 投资逻辑变了吗？

### 7. 巴菲特-芒格协同
- **芒格的补充**: 反向思考、"不投清单"、多元思维模型
- **配合方式**: 
  - 芒格负责"说不"（参谋角色）
  - 巴菲特负责"说是"（交易官角色）
