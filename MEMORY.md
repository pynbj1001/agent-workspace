## 重要决策

### 2026-03-03 - 多 Agent 协作策略
**决策**: 复杂任务优先使用多 Agent 协作模式

**原因**:
- 并行处理效率更高 (5 分钟 vs 15 分钟)
- 专业分工质量更好
- 7 个 Agent 已配置，应该充分利用

**适用场景**:
- 复杂研报 (研究 + 分析 + 写作)
- 公众号发布 (研究 + 写作 + 发布)
- 多股票监控 (每个 Agent 监控几只)
- 跨时区工作 (24 小时轮班)

**Agent 分工**:
- 🎯 总指挥 (main) - 任务分解派发
