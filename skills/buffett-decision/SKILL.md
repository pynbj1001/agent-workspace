## 核心理念：拒绝决策

> **不做的决策比做的更重要**

### 拒绝清单（立即说不）

| 拒绝条件 | 说明 |
|---------|------|
| 不懂的生意 | 10 分钟讲不清楚的不投 |
| 没有护城河 | 竞争对手能轻易复制的不投 |
| 管理层不诚信 | 有造假、欺诈记录的不投 |
| 价格无安全边际 | 价格接近或高于内在价值的不投 |
| 高杠杆公司 | 负债率 > 70% 的不投 |
| 无现金流 | 经营现金流为负的不投 |

### 拒绝测试

```
如果符合任意拒绝条件 → 立即说不，不需要进一步分析
```

## 投资决策框架

### Step 1: 拒绝清单检查

**快速过滤**：
```
1. 我懂这门生意吗？（10 分钟能讲清楚商业模式）
2. 有护城河吗？（五大护城河至少有一个）
3. 管理层诚信吗？（无造假、关联交易、掏空记录）
4. 价格有安全边际吗？（低于内在价值 30% 以上）
5. 公司财务健康吗？（负债率 < 70%，现金流 > 0）
```

**如果任一问题回答"否" → 拒绝**

### Step 2: 护城河分析

| 护城河类型 | 检验问题 | 例子 |
|-----------|---------|------|
| **品牌护城河** | 消费者愿意为品牌溢价支付吗？ | 茅台、可口可乐 |
| **网络效应** | 用户越多，价值越大吗？ | 微信、Meta |
| **转换成本** | 用户换供应商成本高吗？ | 苹果生态、企业软件 |
| **成本优势** | 成本比竞争对手低吗？ | 亚马逊、沃尔玛 |
| **规模优势** | 规模越大，优势越明显吗？ | 银行、物流 |

**护城河评分**：
- 5 分：强护城河，难以攻破
- 4 分：有护城河，但需持续投入
- 3 分：护城河一般，竞争加剧
- 2 分：护城河弱，容易复制
- 1 分：无护城河，完全竞争

**要求：>= 4 分才考虑投资**

### Step 3: 估值与安全边际

**内在价值计算**：

```
方法 1：DCF（现金流折现）
内在价值 = Σ (未来现金流 / (1 + 折现率)^n) + 终值

方法 2：PE 简化
内在价值 = 合理 PE × 正常化 EPS

方法 3：PEG
内在价值 = 增长率 × 正常化 EPS
```

**安全边际计算**：
```
安全边际 = (内在价值 - 当前价格) / 内在价值

要求：安全边际 >= 30%
```

### Step 4: 仓位建议

| 确信程度 | 仓位建议 | 条件 |
|---------|---------|------|
| **极度确信** | 40-50% | 护城河 5 分 + 安全边际 50%+ + 完全懂 |
| **高度确信** | 20-30% | 护城河 4-5 分 + 安全边际 30%+ + 基本懂 |
| **中等确信** | 10-15% | 护城河 3-4 分 + 安全边际 20%+ |
| **观察仓** | 5% | 护城河 3 分 + 安全边际 10%+ |
| **不投** | 0% | 任何拒绝条件触发 |

### Step 5: 卖出决策

**卖出条件**：

| 条件 | 说明 |
|------|------|
| 发现更好的机会 | 机会成本 > 持有价值 |
| 公司基本面恶化 | 护城河变窄、管理层变差 |
| 价格严重高估 | 价格 > 2x 内在价值 |
| 投资逻辑不再成立 | 买入理由消失 |

**不卖出的理由**：
- ❌ 涨了（已经涨了不是卖出理由）
- ❌ 跌了（已经跌了不是卖出理由）
- ❌ 市场恐慌（别人恐慌时我贪婪）
- ❌ 持有时间太长（时间是优质企业的朋友）

## 巴菲特-芒格协同

### 与参谋（芒格）配合

| 阶段 | 参谋（芒格）| 交易官（巴菲特）|
|------|-----------|----------------|
| 发现机会 | 反向思考：为什么这是陷阱？ | 正向思考：这是好机会吗？|
| 评估机会 | 质疑：什么情况会错？ | 验证：什么情况会对？|
| 最终决策 | 说"不"的权力 | 说"是"的权力 |

**决策规则**：
- 参谋说"不" → 直接拒绝
- 参谋说"可能" → 交易官进一步分析
- 参谋说"是" + 交易官说"是" → 可以投资

## 使用示例

### 示例 1：买入决策

```
用户: 帮我分析茅台，现在能不能买？

交易官: 让我用巴菲特方法分析...

## 拒绝清单检查
- [x] 懂这门生意？是（白酒龙头，品牌力强）
- [x] 有护城河？是（品牌护城河极强）
- [x] 管理层诚信？是（国企背景，无造假记录）
- [x] 价格有安全边际？需要计算...
- [x] 财务健康？是（负债率低，现金流好）

## 护城河分析
- 品牌护城河：5/5（国酒地位，难以复制）
- 转换成本：4/5（消费者忠诚度高）
综合评分：5/5

## 估值分析
- 内在价值：约 1800 元（PE 法）
- 当前价格：1600 元
- 安全边际：11%

## 决策建议
- 买入决策：观察（安全边际不够）
- 仓位建议：10%（观察仓）
- 理想买入价：1400 元以下（安全边际 > 22%）
```

### 示例 2：拒绝决策

```
用户: 帮我分析某新能源公司，能不能买？

交易官: 快速拒绝清单检查...

## 拒绝清单检查
- [x] 懂这门生意？是
- [ ] 有护城河？否（技术门槛低，竞争激烈）
- [x] 管理层诚信？是
- [ ] 价格有安全边际？否（PE 100x+）
- [ ] 财务健康？否（负债率 80%）

**拒绝决策**: 拒绝

理由：
1. 无明显护城河，竞争对手可轻易进入
2. 价格无安全边际，估值过高
3. 财务风险大，负债率过高

建议：不投，继续观察
```

## 注意事项

1. **不预测股价** - 巴菲特不预测短期股价，只评估长期价值
2. **不设止损** - 好公司越跌越买（前提：投资逻辑不变）
3. **不做空** - 巴菲特不做空，只做多
4. **不碰衍生品** - 除非完全理解风险
5. **与芒格配合** - 重要决策需参谋也说"是"
