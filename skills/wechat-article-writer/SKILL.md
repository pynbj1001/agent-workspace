# WeChat Article Writer - 公众号写作技能

## 核心能力

### 1. 爆款标题生成
- **数字型**：`3 个信号表明...`、`90% 的人都不知道...`
- **悬念型**：`为什么...？真相是...`
- **对比型**：`从...到...，他只做对了一件事`
- **热点型**：结合时事热点
- **情绪型**：激发好奇、焦虑、共鸣

### 2. 文章结构
```
标准结构：
- 开头（100-200 字）：钩子 + 痛点 + 预告
- 正文（3-5 部分）：小标题 + 论据 + 案例
- 结尾（100-200 字）：总结 + 升华 + 行动号召

爆款结构：
- SCQA 模型：情境 - 冲突 - 问题 - 答案
- 金字塔原理：结论先行 + 以上统下
- 故事模型：英雄之旅 + 情感共鸣
```

### 3. 写作风格
- **通俗易懂**：避免专业术语，用大白话
- **节奏感**：短句 + 长句交替
- **画面感**：具体场景 + 细节描写
- **互动感**：提问 + 引导思考

### 4. 排版优化
- 段落长度：3-5 行一段
- 重点加粗：**关键信息**
- 适当留白：增强可读性
- 表情符号：适度使用 emoji

## 使用方法

```bash
# 生成标题
python3 scripts/article_writer.py --mode titles --topic "人工智能投资机会"

# 生成大纲
python3 scripts/article_writer.py --mode outline --topic "中东局势对油价的影响"

# 完整文章
python3 scripts/article_writer.py --mode full --topic "特斯拉交付量超预期" --style "分析"

# 润色文章
python3 scripts/article_writer.py --mode polish --file draft.md
```

## 文章类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 深度分析 | 逻辑严密、数据支撑 | 行业研究、投资分析 |
| 热点评论 | 观点鲜明、时效性强 | 时事解读、事件点评 |
| 干货分享 | 实用性强、可操作性 | 教程、方法论 |
| 故事叙述 | 情感共鸣、代入感强 | 人物专访、品牌故事 |
| 清单汇总 | 结构清晰、便于传播 | 资源整理、工具推荐 |

## 输出格式

```markdown
# 标题（20-30 字，吸引眼球）

> 导语/摘要（50-100 字，概括核心）

## 小标题 1

正文内容...

**重点内容加粗**

## 小标题 2

正文内容...

---

**结语**

行动号召/思考题
```

## 配置

在 `config.json` 中自定义：
- 目标受众（专业/大众）
- 文章长度（短/中/长）
- 风格偏好（严肃/活泼）
- 行业领域
