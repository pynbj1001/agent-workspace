# 📰 每日新闻晨报技能

自动抓取多个平台的热门内容，生成格式化的新闻晨报。

## 使用方法

### 直接运行
```bash
python3 /root/.openclaw/workspace/skills/news-daily/news_daily.py
```

### 在对话中使用
```
"来份新闻晨报"
"今天有什么新闻"
"生成每日新闻"
```

## 支持的新闻源

| 来源 | 类型 | 状态 |
|------|------|------|
| 虎嗅 | 商业/科技 | ✅ |
| 少数派 | 数字生活 | ✅ |
| 36Kr | 科技财经 | ⚠️ API 变更 |
| GitHub Trending | 开源项目 | 🔄 待添加 |
| Hacker News | 科技新闻 | 🔄 待添加 |

## 输出示例

```markdown
# 📰 每日新闻晨报

**时间：** 2026-02-26 21:59:09 | 周四

## 🐯 虎嗅 - 商业资讯
🔥 1. [新闻标题](链接)
   > 摘要内容...

## 📱 少数派 - 数字生活
⭐ 1. [新闻标题](链接)
   > 摘要内容...
```
